# vue-calculator
Simple and clean calculator with keyboard support 

## Demo
https://thanoseleftherakos.github.io/vue-calculator

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
